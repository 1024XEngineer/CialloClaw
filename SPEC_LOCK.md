# CialloClaw SPEC LOCK

本文件是当前项目的最高优先级实现约束。

若当前实现、历史补丁、临时判断与产品定义文档或本文件冲突：**以本文件与产品定义文档为准**。

## 总原则

1. `orb / nudge / chat / settings` 是 **4 个独立原生窗口**
2. 这 4 个窗口不是一个页面里的 4 个区域
3. 不允许把 `nudge / settings / chat` 的内容继续塞进主窗口
4. `orb` 是入口，不是内容承载窗口
5. `nudge` 是提醒，不是主承接窗口
6. `chat` 是主承接窗口，不是设置窗口
7. `settings` 是低频管理窗口，不参与主任务流

## 核心产品原则

- 桌面常驻
- 低打扰主动协助
- 先提示，再确认，后执行
- 长期协作感
- 不是重平台
- 不是工作区系统
- 不是命令行工具

## 窗口职责锁定

### 1. orb window

职责：

- 只负责常驻入口
- 只负责状态表达
- 只负责把用户导向其他窗口
- 不承载大段内容、任务详情、设置页、确认区

规则：

- 应用启动后立即可见
- chat 打开时 orb 仍然存在
- 只有“隐藏”或“退出”才不可见
- 左键点击 orb：打开或聚焦 chat
- 右键点击 orb：打开自定义菜单（设置 / 隐藏 / 退出）

硬限制：

- orb 必须是独立悬浮实体
- 可拖动
- 无矩形窗口感
- orb 外围不能有矩形留白、矩形阴影、矩形背景、矩形边框
- 若阴影无法跟随球体轮廓，则去掉阴影
- orb 不是主窗口里的一个组件，也不是页面图标

### 2. nudge window

职责：

- 只负责低打扰提醒
- 只负责告诉用户“这里有一个可承接机会”
- 不负责完整任务处理、长对话、复杂设置

规则：

- 由系统识别到机会时出现
- 默认出现在 orb 附近
- orb 在左侧时，nudge 出现在 orb 右边
- orb 在右侧时，nudge 出现在 orb 左边
- 点击“查看”后：打开或聚焦 chat
- 点击“稍后”后：nudge 消失，chat 不打开
- 点击“忽略”后：nudge 消失，本次提醒结束
- 可自动消失
- nudge 消失不影响 orb / chat

硬限制：

- 不要把 nudge 做进 chat 页面右侧
- 不要把 nudge 做成“轻提示记录”固定板块
- 不要让 nudge 长期占据一个大矩形区域

### 3. chat window

职责：

- 唯一主承接窗口
- 负责展示详细上下文、结果、解释、草稿、下一步建议、确认动作

规则：

- 左键点击 orb 时打开
- 点击 nudge 的“查看”时打开
- 若已打开，则聚焦而不是重复创建
- chat 打开时 orb 仍然显示
- chat 关闭时只隐藏 chat，不影响 orb

内容边界：

- 可以有消息流、摘要结果、下一步建议、确认动作
- 不能塞 settings
- 不能承载 nudge
- 不能伪装成整个应用唯一窗口

硬限制：

- 不要再把 chat 做成承载所有区域的大主页面
- 不要把 orb / nudge / settings 并进 chat
- 删除被错误塞进 chat 的固定“轻提示记录”与固定“确认区”之类内容

### 4. settings window

职责：

- 低频管理
- 只处理设置、偏好、主题、权限说明、mock 说明、关于页
- 不参与主任务承接

规则：

- 只能通过 orb 右键菜单里的“设置”打开
- 或托盘菜单中的“设置”打开
- 不能默认常驻显示
- 必须是独立窗口
- 可以与 chat 同时存在

硬限制：

- settings 和 chat 平级
- settings 不是 chat 的 tab
- settings 不是 chat 的抽屉

## 固定交互链路

### 链路 A

orb 常驻
→ 系统识别到机会
→ 在 orb 附近弹出 nudge
→ 用户点“查看”
→ 打开 chat 承接详细内容
→ 用户确认动作
→ 结果反馈
→ orb 继续常驻

### 链路 B

orb 常驻
→ 用户左键点击 orb
→ 打开或聚焦 chat
→ 在 chat 中主动发起总结 / 解释 / 提醒等操作

### 链路 C

orb 常驻
→ 用户右键 orb
→ 菜单出现：设置 / 隐藏 / 退出
→ 点击设置
→ 打开 settings window

## 当前返工范围

只允许修改：

- 多窗口创建与拆分
- orb window 配置与样式
- 右键菜单逻辑
- chat / settings / nudge 的独立窗口化

禁止：

- 重构 Go sidecar
- 重做全项目

## 当前验收标准

1. 右键 orb 不会出现系统默认菜单
2. orb 外围没有矩形窗口感
3. orb 阴影不再是矩形阴影；做不到就无阴影
4. chat 是独立窗口
5. settings 是独立窗口
6. nudge 是独立窗口
7. orb 风格明显更接近简约可爱像素风
