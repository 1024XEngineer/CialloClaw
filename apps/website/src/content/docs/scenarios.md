# 使用场景

## 总结网页

```text
打开网页 → 长按悬浮球 → 说“帮我总结当前页面”
```

适合阅读文章、文档、公告和资料时使用。

------


## 翻译或解释文本

```text
选中一段文字 → 点击悬浮球 → 让它解释或翻译
```

适合处理外文内容、专业术语、代码片段和复杂说明。

------


## 分析文件

```text
拖入文件 → 补一句要求 → 发送
```

例如：

```text
帮我总结这份文档的重点
```

------


## 解释报错

```text
选中报错内容 → 点击悬浮球 → 让它分析原因
```

适合终端报错、网页异常、程序错误和配置问题。

------
