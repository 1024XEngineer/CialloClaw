# 快速上手

## 下载与启动

### 下载

你可以选择两个版本：

| 版本   | 适合人群              | 说明                         |
| ------ | --------------------- | ---------------------------- |
| 最新版 | 普通用户              | 推荐下载，安装后即可使用     |
| 先行版 | 开发者 / 早期体验用户 | 从源码启动，可以体验最新功能 |

### 下载入口：

- [下载最新版（GitHub Releases，latest 当前指向 V0.3.0）](https://github.com/1024XEngineer/CialloClaw/releases/latest)
- [下载先行版（源码仓库）](https://github.com/1024XEngineer/CialloClaw)

------

## 最新版安装

点击「下载最新版」后，根据安装指引一步步完成即可。

安装完成后，启动 CialloClaw。

如果启动成功，你会在桌面上看到悬浮球。

------

## 先行版安装

先行版适合想体验最新功能，或者希望从源码运行 CialloClaw 的用户。

在开始前，请先准备好：

```text
Node.js
pnpm
Git
Rust / Tauri 开发环境
```

### 第一步：获取源码

你可以用两种方式获取先行版源码。

#### 方式一：使用 Git 克隆仓库

打开终端，执行：

```cmd
git clone https://github.com/1024XEngineer/CialloClaw.git
```

然后进入项目文件夹：

```cmd
cd CialloClaw
```

#### 方式二：下载 ZIP

进入 GitHub 仓库页面，点击：

```text
Code → Download ZIP
```

下载完成后，解压 ZIP 文件。

然后进入解压后的 `CialloClaw` 文件夹。

------

### 第二步：安装依赖

在 `CialloClaw` 文件夹根目录右键，打开第一个终端。

执行：

```cmd
pnpm install
```

等待依赖安装完成。

------

### 第三步：启动本地服务

在第一个终端里继续执行：

```cmd
pnpm dev:service
```

执行后不要关闭这个终端。
它需要保持运行。

------

### 第四步：启动桌面端

回到 `CialloClaw` 文件夹根目录。

再次右键，打开第二个终端。

执行：

```cmd
pnpm --dir apps/desktop exec tauri dev
```

等待片刻后，CialloClaw 会自动启动。

如果启动成功，你会在桌面上看到悬浮球。

------

### 先行版启动流程总结

```text
1. 获取源码
2. 进入 CialloClaw 根目录
3. 执行 pnpm install
4. 第一个终端执行 pnpm dev:service
5. 保持第一个终端运行
6. 第二个终端执行 pnpm --dir apps/desktop exec tauri dev
7. 等待桌面端启动
```

第一次启动时，CialloClaw 会自动进入新手指引。

跟随指引操作，即可初步了解悬浮球、语音、工作台和控制面板。

------

## 配置模型

CialloClaw 需要先配置模型，才能正常回复。

### 打开控制面板

在系统托盘中找到 CialloClaw 图标。

右键点击图标，选择：

```text
控制面板
```

进入控制面板后，点击左侧：

```text
模型与安全
```

### 填写模型信息

在右侧填写模型配置。

注意：

```text
Provider 可以自命名。
其余字段需要填写正确的模型服务信息。
```

填写完成后，点击：

```text
测试连接
```

测试成功后，再点击：

```text
保存设置
```

到这里，模型配置就完成了。

------

## 尝试使用

完成配置后，可以先试三个最简单的动作。

### 输入一句话

点击悬浮球下方的输入框，输入：

```text
你好，介绍一下你能做什么
```

发送后，CialloClaw 会在气泡中回复。

------

### 长按语音

长按悬浮球，说出你的需求。

例如：

```text
帮我总结当前页面
```

手势说明：

```text
长按：开始说话
长按后上滑：锁定通话
长按后下滑：取消本次输入
```

------

### 双击打开工作台

双击悬浮球，可以打开工作台。

工作台用于查看任务、结果、记录和更多设置。

------

## 常用动作速览

你可以通过这些方式和 CialloClaw 互动：

| 动作              | 用途                |
| ----------------- | ------------------- |
| 长按              | 语音协作            |
| 长按后上滑 / 下滑 | 锁定通话 / 取消输入 |
| 悬停              | 输入一句话          |
| 双击              | 打开工作台          |
| 选中文本后点击    | 处理选中内容        |
| 拖入文件          | 分析或总结文件      |

刚开始只要记住：**长按说话、悬停输入、双击打开工作台**。
