# CialloClaw 是什么？

CialloClaw 是一款常驻桌面的 Agent 助手。

它会以悬浮球的形式停留在桌面边缘，在你阅读网页、处理文件、查看报错或整理任务时，帮你理解内容、生成草稿、分析文件和推进下一步。



你可以把它理解成：

> 一只在桌面上随时待命的小助手。
> 需要时叫它，不需要时它安静待着。

CialloClaw 的特点：

- **桌面常驻**：通过悬浮球随时唤起
- **轻量协作**：不用打开复杂界面，也能直接提问
- **上下文感知**：可以结合当前页面、选中文本或文件理解需求
- **低打扰**：默认安静待机，需要时再出现
- **可确认**：关键操作会先询问你
- **开源免费**：项目代码公开，欢迎使用和参与贡献
