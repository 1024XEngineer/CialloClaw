# 常见问题

## 工作台白屏了怎么办？

在工作台窗口内右键，选择刷新。

如果仍然无法恢复，可以退出 CialloClaw 后重新启动。

------


## 悬浮球不见了怎么办？

可能是悬浮球贴到了屏幕边缘。

你可以先在屏幕四周找一找。
也可以在系统托盘中右键 CialloClaw 图标，点击：

```text
显示悬浮球
```

------


## 输入后没有回复怎么办？

请先检查：

```text
模型是否已配置
测试连接是否成功
网络是否正常
```

如果你使用的是先行版，还需要确认第一个终端中的服务仍在运行：

```cmd
pnpm dev:service
```

------


## 语音不能用怎么办？

请检查：

```text
系统是否允许 CialloClaw 使用麦克风
当前设备是否有可用麦克风
模型配置是否正常
```

------


## 先行版启动失败怎么办？

可以按顺序检查：

```text
是否已安装 pnpm
是否已安装 Rust / Tauri 开发环境
是否在项目根目录执行命令
pnpm install 是否成功
第一个终端是否保持运行
第二个终端命令是否输入完整
```

------


## 最新版和先行版有什么区别？

**最新版** 更稳定，适合普通用户。

**先行版** 更新更快，适合开发者和早期体验用户，可能会遇到未完成或不稳定的功能。

------


## CialloClaw 是免费的吗？

CialloClaw 本身开源免费。

如果你使用第三方模型服务，可能会产生模型调用费用。具体费用取决于你配置的模型和服务提供方。
