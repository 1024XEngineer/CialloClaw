# Security & Privacy

CialloClaw does not execute everything automatically.

When a task involves files, commands, system state changes, or high-risk operations, it will ask you first. You can confirm what needs to be done, then decide whether to proceed.

You should know:

- Simple Q&A and summarisation can usually be done directly
- Confirmation is requested before critical operations
- The task process is recorded
- Results can be viewed in the workspace
- Tasks can be interrupted if needed
- Model keys are not displayed on ordinary pages
