# Meet the Workspace

How to open:

```text
Double-click the floating ball
```

The workspace is CialloClaw's main interface.

Here you can view:

- Current tasks
- Task history
- Generated results
- Security records
- Model settings
- Memory and task inspection related content

In daily use, you do not need to keep the workspace open.
Most lightweight operations can be done directly through the floating ball.

The workspace is more suitable to open when:

```text
Checking task progress
Viewing generated results
Reviewing what CialloClaw has done
Changing settings
Handling exceptions or failed tasks
```
