# Meet the Workspace

## Dashboard

The dashboard is your central hub. It organises your tasks, notes, memory and safety settings in one place.

### Home View

The home view shows a summary of your current activity:
- Recent and active tasks
- Upcoming reminders and deadlines
- Quick-access module shortcuts
- System status indicators

### Tasks Module

The tasks module lists all your current and past tasks. Each task card shows its status, priority, and a preview of the result. You can drill into any task to see its full conversation history, delivery results, and associated artifacts.

### Notes Module

The notes module keeps nearby notes, future arrangements, and recurring reminders grouped for quick conversion into formal tasks. You can create new notes from scratch or convert existing notes into tasks with a single click.

### Memory Module

CialloClaw remembers your preferences, frequently used tools, and important context from past conversations. The memory module lets you review and manage what CialloClaw remembers about you.

### Safety Module

The safety module gives you full visibility into CialloClaw's operation:
- Review pending authorisation requests
- View audit logs of all actions taken
- Configure risk thresholds
- Manage recovery points for rollback

## Control Panel

The control panel is where you manage:
- General settings (language, theme, voice)
- Floating ball behaviour (position, size, auto-snap)
- Memory policies (retention, summary intervals)
- Task automation (inspection, file watching)
- Model configuration (provider, API keys)
- About and system info
