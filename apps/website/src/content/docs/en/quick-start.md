# Quick Start

## Download and Launch

### System Requirements

- Windows 10 or later (64-bit)
- 8 GB RAM (16 GB recommended)
- 2 GB free disk space

### Installation Steps

1. Download the latest installer from the [GitHub Releases](https://github.com/1024XEngineer/CialloClaw/releases) page.
2. Run the installer and follow the setup wizard.
3. After installation, CialloClaw will appear as a floating ball on your desktop.

## Configure Models

CialloClaw requires a language model to function. You can configure it in the settings panel:

1. Right-click the floating ball and select **Settings**.
2. Go to the **Models** tab.
3. Select your preferred provider and enter your API key.
4. The system supports multiple model providers, including OpenAI, Anthropic, and compatible API services.

## Basic Usage

Once configured, you can interact with CialloClaw through:

- **Voice input**: Speak directly to the floating ball
- **Text selection**: Select text on your screen and CialloClaw will offer to help
- **File drag-and-drop**: Drag a file onto the floating ball to begin working with it
- **Right-click menu**: Right-click the ball for quick actions

## Your First Task

Try selecting a paragraph of text on your screen, then click the prompt button that appears next to the selection. CialloClaw will analyse the content and offer to turn it into a task or provide assistance.
