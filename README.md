# Agent 仪表盘原型

网页原型，用于检查信息架构和边界情况。

## 快速开始

直接用浏览器打开 `index.html` 即可运行。

## 页面结构

| 页面 | 职责 |
|------|------|
| 任务 | 只看正在被 agent 推进的工作（正在进行/暂停/等待授权/失败） |
| 待办 | 只看未来安排（近期要做/后续安排/重复事项/已结束） |
| 摘要 | 高层检查计划纰漏（健康度/风险提示/关键变化/快捷跳转） |
| 镜子 | 历史对话概要 + 日志查看 |

## 核心交互

- **自动轮播**：5 秒切页，hover 主舞台或打开详情时暂停
- **手动切页**：左右按钮或顶部导航
- **卡片操作**：点击卡片打开详情，Esc 或点空白关闭
- **任务修改**：不是弹窗表单，而是暂停后回到悬浮球下方输入框
- **待办转任务**：近期要做详情页点击"开始执行并生成任务"

## Mock 边界覆盖

- 任务：正在进行 / 暂停 / 等待授权 / 失败 / 已完成 / 已取消 / 已结束 3/7/更多
- 待办：近期要做 / 后续安排 / 重复事项规则 / 重复实例 / 逾期 / 完成 / 取消 / 提前 / 恢复
- 摘要：逾期风险 / 时间冲突 / 连续漏做 / 权限阻塞 / 任务失败 / 同步问题
- 镜子：历史概要 / 记忆引用 / 日志追踪 / 跳本地路径（mock 提示）

## 技术限制

- 纯前端，无后端
- 全部 mock 数据，无真实存储
- 无外部依赖
