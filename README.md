# Desktop AI Assistant Prototype

A React-based desktop AI assistant prototype built with Vite, TypeScript, and Vitest.

## Tech Stack

- **React** 18.3.1
- **Vite** 6.3.5
- **TypeScript** 5.8.3
- **Vitest** for unit tests
- **Playwright** for e2e tests

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Run unit tests
npm run test

# Run e2e tests
npm run test:e2e
```

## Project Structure

- `src/app/` - Application components
- `src/styles/` - Stylesheets
- `src/test/` - Test utilities
- `tests/` - E2E tests
- `dist/` - Build output
