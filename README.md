# Agent Task Stage - 原型演示

> 桌面级通用 Agent 任务舞台原型，单文件 HTML，双击即可运行。

## 启动方式

### 最简启动
直接双击打开 `index.html`，无需任何构建步骤：

```bash
# macOS
open index.html

# Windows
# 直接双击 index.html，或：
start index.html

# Linux
xdg-open index.html
```

### 开发调试
如果你想修改代码并实时预览：

```bash
# 使用任意静态服务器
npx serve .
# 或者
python -m http.server 8000
```

然后访问 `http://localhost:8000`（或 3000）

## 核心功能

### 1. 任务舞台（默认首页）

- **主任务卡片**：占据视觉主位的核心任务，显示当前进度、阶段、阻塞状态
- **并行任务**：其他 5 个固定槽位的任务卡片，位置固定、不随内容变化
- **折叠机制**：超出 6 个的任务自动进入折叠任务簇，点击可展开查看
- **任务详情**：点击任意任务卡 → 进入详情态，显示阶段轨迹、产出预览、阻塞原因、引用依据
- **动作入口**：右侧动作脊支持修改目标、补充信息、授权继续、暂停/继续、重试、继续对话

### 2. 工作总览（第二页）

- **日报**：当天进度总览，关键完成项、阻塞、等待用户事项
- **待办三栏**：已完成 / 未完成 / 未开始，结构化任务追踪
- **讨论沉淀**：最近与 Agent 探讨过的内容精华提炼

### 3. 协作镜像（第三页）

- **当前协作理解**：Agent 对用户工作方式的理解
- **记忆簇**：反复出现的约束和偏好
- **上下文/权限概览**：常用的页面和权限

## 页面切换

- **自动轮播**：每 5 秒自动切换到下一页
- **鼠标悬浮暂停**：鼠标停在舞台区域时，轮播暂停；移开后恢复
- **手动切换**：点击左右两侧圆按钮，或使用 ← → 键盘
- **页码指示器**：底部小点指示当前页

---

## 交互细节

| 操作 | 行为 |
|------|------|
| 点击任务卡 | 进入任务详情态 |
| 点击空白区域 | 关闭详情/展开层，返回舞台 |
| 按 Esc | 关闭所有弹层 |
| 按 ← → | 切换上一页/下一页（详情关闭时） |
| 点击折叠任务簇 | 展开任务选择面板 |
| 点击待办/摘要/画像页卡片 | 展开更详细的内容面板 |

## 技术约束

- **零依赖**：纯原生 HTML + CSS + JavaScript，无 npm、无打包
- **离线可用**：所有 mock 数据直接写在页面内，无外部 CDN、无远程字体
- **响应式适配**：主要适配 1440×900 和 1920×1080

---

## 数据替换

所有 mock 数据都在 `<script>` 开头的区域：

```javascript
// 任务数据
const TASKS = [...]

// 日报数据
const DAILY_SUMMARY = {...}

// 讨论沉淀
const CONVERSATION_DIGESTS = {...}

// 协作镜像
const MIRROR_DATA = {...}

// 待办页补充
const TODO_VIEW = {...}
```

后续接入真实后端时，只需替换这些数据对象。
