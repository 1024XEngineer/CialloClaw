# CialloClaw

Desktop agent with low-friction interaction.

## Quick Start

```bash
npm install
npm start
```

## Project Structure

- `main.js` - Electron main process
- `preload.js` - Preload script for secure IPC
- `index.html` - Chat interface
- `dashboard.html` - Dashboard interface
- `claw.html` - Claw interface
- `styles.css` - Global styles

## Tech Stack

- Electron
- HTML/CSS/JS
