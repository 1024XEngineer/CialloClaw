# CialloClaw Demo3

桌面端原型 - Wails + Go + HTML/CSS/JS

## 快速启动

```bash
# 1. 安装前端依赖
cd frontend
npm install

# 2. 构建前端资源
npm run build

# 3. 回到根目录，运行 Wails 应用
cd ..
wails dev
```

或者直接运行编译好的 exe（在 `build/windows/` 目录下）。

## 核心交互

- **双击潮汐核** → 展开悬浮协作窗（上划 = 主处理）
- **左滑来源层** → 查看当前来源、为什么命中、关联任务/记忆
- **顶部模式切换** → 看懂 / 改写 / 推进
- **点击结果卡** → 展开更多内容
- **双击协作窗标题栏** → 折叠/展开

## 功能概览

| 模块 | 说明 |
|------|------|
| 潮汐核 | 中心交互点，提供单一轻量状态（待推进项数） |
| 悬浮协作窗 | 主工作区，显示模式结果、操作按钮、追问输入 |
| 来源层 | 左滑展开，显示当前来源对象、关联任务/记忆、历史来源 |
| 桌面卡片 | 4 张模拟卡片：网页片段、文档卡片、任务现场、报错窗口 |
| 任务/记忆 | 静态模拟数据，展示关联逻辑 |
| Toast 提示 | 轻量操作反馈 |

## 技术栈

- **运行时**: Wails (Go + WebView)
- **前端**: 原生 HTML/CSS/JS（无框架）
- **构建**: npm + build.mjs

## 目录结构

```
demo3/
├── frontend/
│   ├── app.js      # 状态机、渲染逻辑、交互处理
│   ├── styles.css # 样式（协作窗、来源层、主题）
│   ├── index.html # HTML 结构
│   └── dist/      # 构建产物
├── main.go        # Go 后端入口
├── app.go         # Wails 应用配置
└── wails.json     # Wails 配置
```
