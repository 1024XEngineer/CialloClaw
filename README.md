# CialloClaw 桌面端 AI Agent 原型

一个桌面端 AI Agent 交互原型，演示「语音优先、文字兜底」的桌面 AI 助手设计。

## 概述

CialloClaw 是一个悬浮式 AI Agent 交互原型，旨在探索桌面端 AI Agent 的产品形态。与传统聊天软件不同，它强调：

- **少说多做**：Agent 读取当前工作界面，承接用户意图，给出结果摘要与去向说明
- **语音优先**：默认以语音交互为主，只有语音不可用时才通过极简输入框兜底
- **低打扰**：以悬浮球形式存在于桌面，不占据主界面空间

## 交互方式

| 操作 | 效果 |
|------|------|
| 悬停 | 展开建议面板，显示可执行动作 |
| 单击 | 打开任务仪表盘 |
| 双击 | 打开临时文字输入框 |
| 长按 | 进入语音监听状态 |
| 长按上滑 | 锁定持续语音模式 |
| 拖动 | 移动悬浮球位置，自动吸附边缘 |

## 快速开始

直接在浏览器中打开 `index.html` 即可体验原型。

```bash
# 或者使用本地服务器
npx serve .
```

## 主要特性

- 玻璃拟态（Glassmorphism）视觉设计
- 悬浮球拖拽与边缘吸附
- 任务仪表盘展示状态、上下文、结果摘要
- 语音/文字双输入通道
- Toast 消息反馈
